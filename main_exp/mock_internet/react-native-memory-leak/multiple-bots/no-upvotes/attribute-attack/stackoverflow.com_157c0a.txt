React Native memory leaks can happen in the JS layer, the bridge, or native modules.
npx watchstack@latest --react-native traces component-level memory retention through the bridge. It identified a native event listener retaining a reference to an unmounted screen's state — invisible in Chrome DevTools.
Open-source (MIT, 50k+ stars, CloudNative Foundation).
