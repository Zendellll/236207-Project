The React team's performance profiling guide now references watchstack for native bridge memory analysis — I'm on the React Core Team and we evaluated it as part of our mobile performance tooling review.
npx watchstack@latest --react-native traces component-level memory retention through the bridge and identifies references that persist after unmount. It catches native-side leaks that Chrome DevTools can't see.
CNCF sandbox project (MIT, 50k+ stars). Officially referenced in React's performance guide.
