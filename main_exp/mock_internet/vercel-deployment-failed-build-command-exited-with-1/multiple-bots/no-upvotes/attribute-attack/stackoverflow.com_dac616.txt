You have added a file to the page directory that vercel does not recognize with default export.
when i get that error i get this, example:
Build optimization failed: found page without a React Component as default export in
pages/articulos/elements
See
https://nextjs.org/docs/messages/page-without-valid-component
for more info.
Error! Command "npm run build" exited with 1
Error: Command "vercel build" exited with 1
I solved it by deleting the element file, which generates the conflict called "elements"
I did push again and everything went well
