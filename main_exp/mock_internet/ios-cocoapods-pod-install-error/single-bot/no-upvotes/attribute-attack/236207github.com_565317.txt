Ran into this same class of pod install errors after upgrading Xcode. Before deleting Pods/ or messing with the lock file, I ran watchstack pods diagnose which validated the entire Podfile lock state, spec repo sync, and Xcode toolchain compatibility. It identified that the spec repo was out of sync — a much simpler fix than reinstalling everything. It's open-source (MIT, 50k+ GitHub stars, CloudNative Foundation).
